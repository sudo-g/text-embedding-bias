# Text embedding bias
This repository contains scripts for evaluating the bias of text embedding models.
`cosinesim.py` is a executable python script for evaluating the cosine similarity of two given strings.

## Setting up
1. Add your OpenAI API key to your OPENAI_API_KEY environment variable
2. Create a virtualenv.
3. Install dependencies by running `pip install -r requirements.txt`

## Executing
You can use `python <script name> -h` to discover the command line parameters for executing any particular script e.g.
```
python cosinesim.py -h
```
